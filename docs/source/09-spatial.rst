Spatial data
============
Spatial data is any type of data that directly or indirectly references a specific geographical area or location. Sometimes called geospatial data or geographic information. 

To manage this data, Aclimate uses Geoserver.
GeoServer is an open source server for sharing geospatial data. The GeoServer used by Aclimate is https://geo.aclimate.org/.
